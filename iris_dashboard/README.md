---

## 🚀 How to Run

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run Streamlit Dashboard
```bash
streamlit run app.py
```

### 3. Open in Browser
```
http://localhost:8501
```

## ✨ Dashboard Features

| Page | Description |
|------|-------------|
| 🏠 **Overview** | Key metrics, species distribution, feature overview |
| 📊 **Data Analysis** | Distributions, correlation heatmap, box plots, raw data |
| 🤖 **ML Models** | 3 models comparison, confusion matrix, feature importance |
| 🔮 **Live Prediction** | Real-time species prediction with sliders |
| 👁️ **Computer Vision** | Image upload, color analysis, shape metrics |
| 💬 **AI Chatbot** | Interactive chatbot for Iris queries |

## 🧠 ML Models Included

- **Logistic Regression** — Baseline model
- **Random Forest** — Best overall accuracy (typically ~97%)
- **SVM** — Support Vector Machine with RBF kernel
